    ### Ethereum Blockchain Todo App

### Commands to Execute

#### Install Truffle Globally using npm

```bash
$ npm install -g truffle@^5.0.2

```

#### Create Project Directory

```bash

$ mkdir eth-todo-app
$ cd eth-todo-app


```

#### Initialize Project
```bash
$ truffle init
```

#### Compile Contracts
```bash
$ truffle compile
```

#### Migrate contracts
```bash
$ truffle migrate
```

#### Run the console
```bash
$ truffle console
```
